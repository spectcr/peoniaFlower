# 🌸 Peonía Interactiva en 3D

Una animación interactiva de una peonía desarrollada con [p5.js](https://p5js.org/). El proyecto corre directamente en el navegador, siendo súper liviano y sin requerir instalaciones adicionales.

🚀 **[Ver Demo en Vivo Aquí](https://TU_USUARIO.github.io/TU_REPOSITORIO/)** *(Asegúrate de cambiar este enlace por la URL real de tu GitHub Pages)*

---

## ✨ Qué hace

Esta animación simula el ciclo natural de la flor con toques digitales:
* **Ciclo de vida:** La peonía florece por sí sola, transiciona entre distintas paletas de colores y, eventualmente, se marchita.
* **Interactividad 3D:** El ángulo y la rotación de la flor reaccionan al movimiento de tu mouse.
* **Filtros dinámicos:** Los efectos visuales van rotando de manera automática, alternando entre estilos de **ASCII**, **puntos** y **píxeles**.
* **Glitch Art:** Cada cierto tiempo aparece un efecto de *glitch* espontáneo que distorsiona la imagen.

## 🛠️ Tecnologías utilizadas

El proyecto se mantiene intencionalmente simple y limpio:
* **[p5.js](https://p5js.org/)**: Motor principal para toda la lógica gráfica, manejo del canvas y renderizado 3D.
* **HTML/CSS/JS Vanilla**: Sin frameworks, sin bundlers y sin dependencias pesadas.

## 💻 Cómo usarlo en local

¡Es tan simple como abrir un archivo!
1. Clona o descarga este repositorio.
2. Haz doble clic en el archivo `peonia.html` para abrirlo en tu navegador preferido. 
3. *No necesita servidor local, no necesita npm, no necesita nada más.*
